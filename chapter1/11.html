Cómo usar una media query

Una media query es una condición para la inclusión o no de una regla css.

Puedes incluir una media query directamente en el head.

@media screen and (min-width: 1024px) {
	body {
		font-size: 100%;
	}
}

Mediante una etiqueta link.

link rel="stylesheet" href="wide.css" media="screen and (min-width: 1024px)"

O mediante una sentencia de import

@import url("wide.css") screen and (min-width: 1024px);

La lista de parámetros que podemos introducir en las media query son los siguientes:

width
Ancho del display.
height
Alto del display.
device-width
Ancho del área de renderizado del dispositivo.
device-height
Alto del área de renderizado del dispositivo.
orientation
portrait o landscape
aspect-ratio
Relación de aspecto de la ventana del navegador (p.ej.)
device-aspect-ratio
Relación de aspecto del dispositivo.
color
Número de bits de color de un dispositivo. Un dispositivo de 8 bits devolvería 8, uno sin color 0.
color-index
Número de entradas de color en la tabla de lookup del dispositivo de salida. Por ejemplo @media screen and (min-color-index: 256)
monochrome
Nos permite saber el nº bits por pixel en un dispositivo monocromo.
resolution
Densidad de píxeles en el dispositivo: screen and (resolution: 72dpi) or screen and (max-resolution: 300dpi).
scan
En televisiones nos permite saber si el procesado es progressive o scan
grid
Comprueba si el dispositivo tiene un display basado en grid. Como los teléfonos con un único tamaño de fuente.
